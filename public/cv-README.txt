Place your CV here as `cv.pdf` (i.e. public/cv.pdf).

Anything in this `public/` folder is served from the site root, so the file
will be downloadable at /cv.pdf — which is what the "Download CV" button and
the header "CV" link point to.

If you'd rather use a different filename or format, update `cvPath` in
src/config.ts to match.

You can delete this note once your CV is in place.
